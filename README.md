## Install
Follow The Steps Below!

```bash
> termux-setup-storage
(after that tap on permission)
> apt install git -y
> git clone https://github.com/reidogadobot/shaolin
> cd shaolin
> bash install.sh
```

## Usage

```bash
> npm start
```
